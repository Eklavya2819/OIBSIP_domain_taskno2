# Eklavya Pandey – Portfolio Website

## Overview

This project is a **personal portfolio website** built using **HTML and CSS**. The website presents the professional profile of **Eklavya Pandey**, a B.Tech Computer Science student interested in **Data Science, Machine Learning, and Web Development**.

The portfolio highlights personal information, technical skills, experience, and provides a downloadable resume.

---

## Features

* Fixed **navigation bar** with smooth scrolling
* **About section** introducing the developer
* **Skills section** displaying technical abilities
* **Experience section** showing internship and ambassador roles
* **Resume download button**
* **Responsive layout** for smaller screens
* Clean **dark theme UI**

---

## Technologies Used

* **HTML5** – Website structure
* **CSS3** – Styling and layout design

No external frameworks or libraries are used in this project.

## How to Run the Project

1. Download or copy the project files.
2. Ensure the following files are in the same folder:

   * `index.html`
   * Profile image file
   * `Eklavya_Pandey_Resume.pdf`
3. Open **index.html** in any web browser.

## Responsive Design

The website includes a media query for screens smaller than **768px**, ensuring:

* The about section stacks vertically
* Navigation links wrap properly
